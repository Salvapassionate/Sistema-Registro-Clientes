Librerias a descargar

Instalar librerias en carpeta sistema: 
npm install react react-dom

npm install react-router-dom
npm install react-uuid  

instalar axios
npm install axios

e instala navegador para descargar en disco duro:
npm install file-saver

Instalar en la carpeta server para backend:

instalar la backend de postgresql
npm install pg express

instalar para desbloquear el servidor localhost:3000
npm install cors


Para el servidor backend
npm install express sequelize pg sequelize-cli

express, pg (driver de PostgreSQL para Node.js) y body-parser para manejar las solicitudes JSON.
npm install express pg body-parser
